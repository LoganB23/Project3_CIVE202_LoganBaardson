This project is focused on analyzing and presenting the results for two key data sets that are very prevalent in
the transportation industry. These two data sets are the National Household Travel Survey (NHTS) and the Next
Generation Simulation (NGSIM). The NHTS data set is focused on observing different characteristics of transportation
while the NGSIM data is focused specifically on the behaviors and actions of drivers on the road. The details of the
files in this project can be found below.
